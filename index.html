HTML Structure

Doctype:Specifies the document type and the html version used.

Head:add meta,link,style,script tags here which are not visible to user on screen.

Body:Content here is visible to user on screen.

Attributes:additional information of a tag.

Meta:Specifies metadata(information) of a website like keywords,description,author,char set.They are not visible on website to user,they are used by the browser.
